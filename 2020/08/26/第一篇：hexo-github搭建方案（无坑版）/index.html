第一篇：hexo+github搭建方案（无坑版）

Posted on 2020-08-26

搭建方案整体分为五步：

第一步（准备阶段）

整体搭建过程需要用到：Git、node.js、hexo、github

第二步（node安装）

通过官网地址下载：https://nodejs.org/zh-cn/
下载成功后，进行手动安装，安装成功后（此时npm管理器及node.js都已经安装完成了）
通过终端进行查看（mac版本如下）
    1.sudo su 获取管理员权限
    2.输入mac中的密码
    3.验证通过后，node -v，查看node版本号
    4.npm -v，查看nom版本号

第三步（cnpm）

安装npm，npm install -g
安装成功后，安装博客框架，npm install -g hexo-cli 
创建一个blog文件夹，mikdir blog
进入blog，cd blog 
初始化一个博客，sudo hexo init 
启动这个博客，hexo s
创建我的第一篇文章，hexo n '我的第一篇文章'
清理缓存，hexo clean
生成文章，hexo g

第四步（github）

创建github账号
创建成功后，创建库名，文件名称必须与库名一致。.github.io
装git插件，cnpm install --save hexo-deloyer-git
设置_config.xml，配置deploy（1.type:git、2.repo:仓库全地址、3.branch：master）
设置git账号和用户名（1.git config --global user.name "cxxxb"、2.git config --global user.email "10@qq.com"）
推到远端服务器，hexo d

第五步（修改主题）

在hexo官网或者github上找到喜欢的主题，进行克隆，git clone+'github地址/下载压缩包'
修改config.xml，找到themes文件，修改theme为新主题名称
重新生成，hexo g
推到本地看效果，hexo s
推到github上，hexo d

此时hexo+github搭建博客已经搭建成功了!
